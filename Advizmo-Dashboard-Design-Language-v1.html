Dashboard Design Language

The blueprint for every dashboard Advizmo will ever build

01 Dashboard Philosophy

The dashboard exists to answer five questions. Every element should help answer one of them.

💰
How am I doing?
Hero metric, net worth
⚠️
What needs attention?
Alerts, upcoming bills
🎯
What should I do next?
AI insights, quick actions
📈
How has my position changed?
Charts, KPI changes
💡
What opportunities exist today?
Recommendations

Core Principles

✓
One primary financial insight — The most important number
✓
One primary action — The most important thing to do
✓
Progressive disclosure — Summary first, details on demand
✓
5-second rule — Users understand their situation in 5 seconds

02 Information Hierarchy

Every dashboard should communicate information in this order.

1
Primary Information
Net worth, total portfolio value, available cash
2
Secondary Information
Daily change, monthly performance, account count
3
Supporting Information
Account types, last updated, transaction categories
4
Historical Context
30-day performance, year-to-date returns
5
Future Forecast
Projected growth, estimated returns
6
Recommendations
AI insights, suggested actions

05 Hero Section

The hero displays one primary metric. Never compete for attention.

High Priority

For active investors
Medium

For liquidity-focused users

Financial Health Score
For health-focused users
Low

Hero Rules

✓
One primary metric — Never compete for attention
✓
Large typography — 32px minimum
✓
Clear context — Show change, timeframe, trend
✓
Calm presentation — No flashing, no urgency

06 KPI Language

Standardized KPI presentation. Never display numbers without context.

Portfolio Performance
$124,500.00
+$2,450.00 (2.0%) today
↗ Upward trend

KPI Components

Component Required Description
Label Yes What this KPI measures
Value Yes Current value
Change Yes Change from previous period
Timeframe Yes Period of change
Trend Yes Direction indicator
CTA Optional Call to action

07 Financial Storytelling

The dashboard should tell a story: Yesterday → Today → Tomorrow.

Yesterday
Your portfolio grew by 2.0% this week
→
Today
Because of your recurring investments
→
Tomorrow
Consider increasing your monthly contribution

Storytelling Rules

✓
Cause and effect — Show why things happened
✓
Time progression — Past → Present → Future
✓
Actionable insights — Suggest what to do next
✓
Calm presentation — No urgency, no anxiety
