08 Reduced Motion

All animations must respect the user's motion preferences.

Animations to Remove

  • Page transitions → Instant swap
  • Sheet presentation → Instant appear
  • Modal presentation → Instant appear
  • Toast enter/exit → Instant appear
  • Skeleton shimmer → Static placeholder
  • Spinner rotation → Static spinner
  • Number count → Instant final value
  • Chart animation → Static final state

Animations to Keep

  • Focus ring → Accessibility requirement
  • Loading indicator → Still needed (static)
  • Progress bar → Static fill still shows progress
  • Skeleton → Static placeholder indicates loading
CSS Implementation: @media (prefers-reduced-motion: reduce) {
  *, *::before, *::after {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}

15 Error Recovery

Every error must explain what happened, why, and what the user can do.

Error Message Examples

Connection Lost

Please check your internet connection and try again.

Insufficient Funds

Your checking account has $1,234.56 available. The requested transfer of $5,000.00 exceeds your balance.

Invalid Email Address

Please enter a valid email address (e.g., user@example.com).

Error Message Rules

  • Never blame the user. Say "Something went wrong" not "You made an error."
  • Never use vague language. Say "Network connection lost" not "Error occurred."
  • Always be specific. Say "Insufficient funds" not "Payment failed."
  • Always be actionable. Say "Please try again" not "Try again later."

17 Accessibility Testing

Every component must pass these accessibility checks.

VoiceOver

  • All elements have labels
  • All interactive elements have hints
  • All state changes are announced
  • Reading order is logical
  • No focus traps

Dynamic Type

  • All text scales correctly
  • No text is clipped
  • Layout adapts to larger text
  • All content remains accessible

Keyboard

  • All elements reachable via Tab
  • Tab order is logical
  • All actions achievable via keyboard
  • Focus indicators visible

Contrast

  • Normal text: 4.5:1 minimum
  • Large text: 3:1 minimum
  • Interactive: 3:1 minimum
  • Focus rings: 3:1 minimum
