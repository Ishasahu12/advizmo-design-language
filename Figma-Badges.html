Status dot: 8px default, pulse for active/warning states
Chip selected state must be visually distinct
Use semantic colors: green = success, red = error, amber = warning
Removable chips need clear × affordance
Never use badge for interactive actions
Never use more than 6 color variants
Never mix badge and tag for same purpose
