13 Accessibility

Every animation must have a reduced motion alternative.

Reduced Motion Support

When prefers-reduced-motion: reduce is enabled:

  • All animations are replaced with instant state changes
  • Page transitions become instant swaps
  • Sheet/modals appear instantly
  • Chart animations show final state only
  • Focus rings remain visible
  • Loading indicators remain (static, not spinning)
