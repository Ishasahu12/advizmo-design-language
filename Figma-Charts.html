Rules
All financial values in IBM Plex Sans with tabular nums
Grid lines: gray.200 dashed, never solid
Use gradient fill under line/area charts for depth
Bar chart: green for positive, red for negative
Donut chart: max 5 segments, group rest as "Other"
Sparkline: no axes, no labels, just the trend
Gauge: semicircle (180°), value centered below
Always include legend when multiple series present
Never use 3D effects on charts
Never use pie charts (prefer donut)
Never start Y-axis at non-zero for bar charts
